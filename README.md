# Runmote

Control AI agents on your laptop from your mobile phone. Connect via cloud relay at `runmote-relay.onrender.com`.

## Install

### Linux / macOS
```bash
curl -fsSL https://runmote.dev/install.sh | bash
```

Dev branch:
```bash
curl -fsSL https://runmote.dev/install.sh/dev | bash
```

### Windows
```powershell
powershell -c "irm https://raw.githubusercontent.com/Raza-learner/Runmote/dev/scripts/install.ps1 | iex"
```
